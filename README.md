wdtvmd - Metadata lookup tool for WDTV devices
==============================================

This is a simple tool to generate metadata files for WDTV devices from
TMDB.

Requirements
------------

* python
* tmdb3
* tvdb_api

Overview
--------

     # ./wdtv -t /path/to/Friends/Season 3/Friends S03E10.mkv

or

     # ./wdtv -t -r /path/to/Friends
